# Honest Hammer ∞

Professional end-to-end construction services website for **Honest Hammer**, operating across **Chennai, Hyderabad, and Bangalore**.

## 🏗️ Features

- 🔨 12 core construction services with interactive cards
- 📊 Animated statistics counter
- 📱 Fully responsive (mobile, tablet, desktop)
- 💬 WhatsApp integration for instant quotes
- 🗺️ Google Maps embedded
- ⭐ Customer review ticker
- 📖 Knowledge base with industry insights
- 🎨 Professional construction color scheme

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript
- Font Awesome Icons
